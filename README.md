# mqqikq.github.io

Source for [mqqikq.github.io](https://mqqikq.github.io) — a portfolio landing
page linking to:

- [tiny-httpd](https://github.com/mqqikq/tiny-httpd) — concurrent HTTP/1.1
  server in C (epoll, non-blocking I/O, SO_REUSEPORT, live SSE dashboard,
  Docker demo)
- [flashcards-api](https://github.com/mqqikq/flashcards-api) — spaced-repetition
  flashcards REST API (FastAPI + SQLite + SM-2-lite)

Plain HTML/CSS, no build step, served directly by GitHub Pages from `main`.
